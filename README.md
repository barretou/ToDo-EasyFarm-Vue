<h3 align='center'>About</h3>
<p>Aplicação "To Do List" construída com Vue JS. </br> Apresenta funcionalidades básicas como <b>CRIAR</b> uma nova tarefa, <b>ATUALIZAR</b> uma já existente e <b>DELETAR</b> tarefas</p>

<h3 align='center'>Getting Started</h3>
O framework foi adicionando no HTML via CDN. Então basta fazer o download dos arquivos deste repositório, colocar em uma pasta e abrir o ```index.html``` com o navegador de internet da sua escolha e aproveitar o app!
